# MyWearCalendar

Test in a web browser:
  git clone <repo name>
  
  cd into new repo
  
  npm install
  
  ionic serve (assuming you have ionic cli downloaded)
  
Test for IOS device:
  git clone <repo name>
  
  cd into new repo
  
  npm install
  
  ionic cordova build ios
  
  open project in Xcode
  
  select emulator
  
  build project
